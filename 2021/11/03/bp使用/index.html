Burp Suite使用

1.Target模块

是站点地图，用来显示信息。默认记录浏览器访问的所有页面，并且使用Spider模块扫描后，可以清楚的看到爬虫所爬行的页面及每一个页面的请求头和响应。所有页面也可以通过添加过滤器（Filter）来过滤非目标站点。

具体方法：

将目标站点添加到Scope。在Filter里可以选择”show only in-scope items”，只显示范围内的列表，即可进行过滤。使用可以根据请求类型，MIME类型，HTTP状态等方式进行过滤显示。

2.Spider

完整地枚举应用程序的内容和功能。send to spider 后可以在Target的Site map里看到爬行列表。爬行同时会进行被动漏洞扫描，检测每一个访问过的URL。

3.Scanner

帮助渗透人员发现Web应用程序的安全漏洞。可以针对单一的Url进行测试，可以在拦截到请求后单击“Action”按钮，然后选择“Do an active scan”进行主动扫描。也可以全站扫描，先爬行网站所有的链接。在Target模块里选择需要进行扫描的网站，并单击右键在弹出的快捷菜单中选择“Actively Scan this Host”主动扫描主机。

删除不需要扫描的页面，提高扫描的速度。

配置信息Opition:

Results中可以查看扫描结果和安全信息。

4.Intruder

最重要的是配置Attack Type、程序变量、字典设置。

拦截到网站请求后选择send to Intruder,然后在手工选择“Intruder”->”Target”才可看到目标网址和访问端口。

Intruder的opitions可以看到拦截后的HTTP Request请求，也是我们主要配置的地方，具体配置如下：

positions

（1）Attack Type

攻击类型配置：选择

(2)变量

Burp Suite会自动在请求头中加入许多变量，我们先clean,再add加自己想要的变量。

payloads

（3）字典

Payloads有四个区域：

1.payload sets:

payload set 针对指定变量进行配置。

payload type

2.payload options:

默认是simpl list类型，随payload type设置而变化。

3.payload processing:

为bp最强大的功能之一，可以有效地对字符串进行处理（字典的每一行），可以进行MD5加密，字符串截取、加入前缀、后缀等操作。

4.payload encodeing:

在进行请求时，针对某些字符进行URL转码.

options

（4）option模块

配置请求线程、请求结果集格式等。

配置后单击“Intruder”—>”start attack”进行扫描。

4.辅助模块

（1）decoder解码器

支持对URL、HEX、HTML等格式字符串进行编码或解码操作。

（2）comparer比较器

字符串比较器可以迅速帮我们发现两段字符串中的差异。单击“words”进行比较。

（3）repeater

http请求编辑工具，点击“go”。

（4）乱码

response中文字符往往会乱码，

配置“option”

http message display选择change font选择微软雅黑或宋体来解决中文乱码问题。

6.sqlmap插件

下载python2.7并配置环境，下载salmap有关内容包并配置环境，下载sqlmap.jar
